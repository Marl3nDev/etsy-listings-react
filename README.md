# Etsy Listing React App

Это React TypeScript приложение, которое отображает список объявлений Etsy. Построено с использованием Vite для быстрой разработки.

## Функции

- Отображает карточки товаров с изображениями, заголовками, ценами и количеством на складе
- Фильтрует для показа только активных объявлений
- Форматирование цен в зависимости от валюты (USD, EUR, GBP, CAD)
- Бейджи количества с цветовым кодированием (низкий, средний, высокий)
- Цифровой бейдж для цифровых товаров
- Адаптивная сетка
- Усечение заголовков для длинных (>50 символов)

## Демо

Посмотреть демо: [GitHub Pages](https://marl3ndev.github.io/etsy-listings-react/)

## Установка

```bash
npm install
```

## Запуск приложения

```bash
npm run dev
```

Приложение будет доступно на `http://localhost:5173/`.

## Данные

Данные товаров берутся из `src/data/etsy.json`, содержащего 24 активных объявления Etsy.

## Технологии

- React 18
- TypeScript
- Vite
- CSS
